# Demo Instructions

## How to Test the Chatbot

1. **Open the chatbot in your browser:**
   - Navigate to: http://localhost:8501

2. **Try these example questions:**
   - "What is LangChain?"
   - "How do I install LangChain?"
   - "What are the main features of LangChain?"
   - "How do I use LangChain with OpenAI?"
   - "What is a retrieval chain?"

3. **Observe the following:**
   - ✅ User messages appear on the right
   - ✅ Assistant messages appear on the left
   - ✅ "Thinking..." spinner appears while processing
   - ✅ Responses are contextual from the LangChain docs
   - ✅ Chat history is maintained across questions
   - ✅ Server status indicator shows green when connected

4. **Test additional features:**
   - Click "Clear Chat History" in the sidebar
   - Check the "About" section in the sidebar
   - Verify server status indicator

## Expected Behavior

The chatbot should:
- Respond within 2-5 seconds
- Provide relevant answers based on LangChain documentation
- Maintain conversation context
- Handle errors gracefully (show error messages)
